# 電子密碼鎖 (8051 專案)

## 專案動機
本專案旨在結合 8051 微控制器架構與數位邏輯，設計一套直覺的門禁控制系統，並透過 LED 與蜂鳴器達成狀態反饋。

## 功能說明
- **密碼輸入：** 使用 16 鍵矩陣按鈕進行輸入。
- **狀態回饋：**
  - **正確：** 綠燈亮 + 長音。
  - **錯誤：** 紅燈閃爍 + 警示音。
- **防彈跳機制：** 確保輸入訊號穩定。

## 硬體架構
- **控制器：** 8051 (KT-89S51)
- **輸入：** 4x4 矩陣按鍵
- **輸出：** LED (綠/紅)、蜂鳴器
